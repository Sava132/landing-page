# Modern Developer Portfolio 🚀

This is a personal landing page designed to showcase my identity as a frontend developer. It focuses on modern design trends, clean typography, and a smooth user experience.

## 🚀 Live Demo
[Explore My Portfolio](https://sava132.github.io/landing-page/)

## ✨ Key Features
* **Modern UI:** Implemented with Glassmorphism and dark mode aesthetics.
* **Fully Responsive:** Adapts perfectly to mobile, tablet, and desktop screens.
* **Optimized Performance:** Built with clean, semantic HTML and efficient CSS.

## 🛠️ Tech Stack
* **HTML5** - Semantic structure.
* **CSS3** - Custom properties, Flexbox, and Glassmorphism effects.
* **Git/GitHub** - Version control and deployment.

This project was created as part of my professional identity on the Contra platform to demonstrate my ability to build aesthetic and functional web interfaces from scratch.
